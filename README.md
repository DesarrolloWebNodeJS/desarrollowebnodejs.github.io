Proyecto NodeJS y Ruby
=========================

1. Clonar este repositorio, instalar dependencias y correr el servidor

    ```bash
	git clone git@github.com:fizzvr/fizzvr.github.io.git 
	cd fizzvr.github.io
	npm install
	npm start
	```

1. Hackeando en [http://localhost:3004/](http://localhost:3004/)
